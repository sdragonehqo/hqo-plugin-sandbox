# Hashtag Library

Core hashtag sets to rotate from. Use 3-5 per post: 1 broad reach + 1-2 industry niche + 1-2 topic-specific.

## Broad Reach Tags (pick 1)
- #CommercialRealEstate
- #CRE
- #FutureOfWork
- #WorkplaceExperience
- #PropTech

## Industry Niche Tags (pick 1-2)
- #TenantExperience
- #BuildingTechnology
- #PropertyManagement
- #AssetManagement
- #CREInnovation
- #SmartBuildings
- #WorkplaceStrategy
- #OfficeTrends

## Topic-Specific Tags (pick 1-2, rotate based on post topic)

- #BuildingOperations
- #EmployeeExperience
- #OccupierDemand
- #CREData
- #LeaseUp
- #BuildingAmenities
- #WorkplaceCulture
- #CRELeadership
- #PropertyTech

## Branded Tags (use sparingly, roughly 1 in 4 posts)
- #HqO
- #REX

## Tags to AVOID
- Any residential real estate tags (#RealEstate, #HomesBuying, #Realtor, etc.)
- Overly generic tags (#Business, #Leadership, #Success, #Motivation)
- More than 5 hashtags per post
- Hashtags that don't match the post content
