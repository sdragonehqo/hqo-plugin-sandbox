# HqO Marketplace Plugins

This is the authoritative list of plugins published to the HqO plugin marketplace.
The `plugin-setup` skill uses this to filter installed plugins — only plugins that
appear here will be included in setup runs. This prevents the skill from attempting
to set up pre-installed remote or built-in Claude Code plugins.

**Update this list whenever a new plugin is merged into the marketplace.**

## Published Plugins

| Plugin Name | Description |
|-------------|-------------|
| `cowork-memory` | Saves completed Cowork sessions to local memory |
| `cowork-email-digest` | Digests last 24h of Gmail into memory system |
| `cowork-slack-digest` | Digests last 24h of Slack DMs and channels into memory |
| `cowork-gemini-notes` | Digests last 24h of Google Gemini meeting notes into memory |
| `personal-progress-tracker` | Onboarding, recall, and setup for the cowork memory system |
| `rex-linkedin-expert` | REX Expert LinkedIn thought leadership for HqO employees |
| `hqo-plugin-scaffold` | Developer tool for scaffolding and publishing HqO plugins |
| `cowork-maintenance` | Master onboarding and debugging for all HqO plugins |
