# cowork-demo Changelog

## [1.0.0] — 2026-03-25

### Added
- Initial release.
- `tour` skill: Main entry point — conducts the full interactive demo experience across all four chapters.
- `file-chaos` skill: Creates 20 realistically messy files, reads their contents, and organizes them into clean folders.
- `dashboard` skill: Generates a live, animated HTML capabilities dashboard from scratch.
- `agent-demo` skill: Launches three parallel sub-agents simultaneously and aggregates their findings.
- `slack-chaos` skill: Previews messaging the entire team, schedules the task live, then cancels it before it fires.
